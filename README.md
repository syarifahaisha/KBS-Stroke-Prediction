# KBS-Stroke-Prediction

## How to run
### Library Requirement 
- django
- flask 
- import-ipynb
- imblearn

``` pip install -r requirements.txt ```

### Command Run
go to kbs-env directory
``` cd kbs-env ```
then run the program
``` flask run ```
