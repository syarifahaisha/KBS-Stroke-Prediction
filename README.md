# KBS-Stroke-Prediction

## How to run
### Library Requirement 
- django
- flask 
- import-ipynb
- imblearn


go to kbs-env directory
``` cd kbs-env ```
then install required libraries from requirements.tct
``` pip install -r requirements.txt ```

### Command Run
then run the program
``` flask run ```
